# react-midway-ssr

### 一、技术栈
react、midway.js(基于egg.js)等技术栈的前后端同构应用;
1. react V16.10
2. midway.js V2.0
3. lowdb V1.0（json文件数据库）
4. cheerio V1.0（爬虫技术：编写一个定时器，去github上爬取指定的内容）

### 二、演示地址
[点我进入React ssr](http://47.98.40.154:7001/)

### 三、如何run
```javascript
npm i
// dev
npm start
// prod
npm run prod
```
